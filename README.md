# Fake_News_Detection
Fake News Detection is a Django-based web application that detects fake news using Machine Learning and NLP techniques. It analyzes news text, predicts whether it is real or fake, and provides an easy-to-use interface for reliable news verification with high accuracy.
# 📰 Fake News Detection Using Machine Learning

## 📌 Overview
Fake News Detection is a Django-based web application that identifies whether a news article is **Real** or **Fake** using Machine Learning and Natural Language Processing (NLP). The system preprocesses the input text, extracts important features, and predicts the authenticity of the news using a trained classification model.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Fake_News_Detection.git
```

### 2. Navigate to the Project

```bash
cd Fake_News_Detection
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run Database Migrations

```bash
python manage.py migrate
```

### 6. Start the Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
